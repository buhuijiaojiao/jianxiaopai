## 🪄 功能简介

AI 海投机是简小派推出的 **自动化投递工具**，支持智能筛选职位、自动填写投递信息、智能跳过不匹配岗位。  
当前版本主要支持 **Boss直聘** 平台，并可结合简小派账号进行行为记录与效果追踪。

## 🧠 第二步：导入脚本到浏览器

AI 海投机基于浏览器插件 **Tampermonkey（篡改猴）** 运行。

:::tip 安装指南
1. 打开浏览器（推荐使用 **Chrome** 或 **Edge**）  
2. 前往 [Tampermonkey 官方下载页](https://www.tampermonkey.net/)  
3. 点击安装并启用扩展程序  
4. 安装完成后，浏览器右上角扩展程序会出现一个黑色的“猴子脸”图标
:::

### 📥 导入脚本
1. 点击篡改猴图标 → **“添加新脚本”**  
![添加脚本](..\static\img\autosend\autosend-step1.png)
2. 将你收到的脚本拖拽进去  
![拖拽脚本](..\static\img\autosend\autosend-step2.png)
3. 点击安装/重新安装，脚本 ✅

:::warning 注意

需要启用浏览器扩展的开发者模式，脚本才能生效
:::

---

## 🚀 第三步：开始自动投递

安装完成后，打开 **Boss直聘** 网站并登录。  
点击**职位**
![职位](..\static\img\autosend\autosend-step3.png)
此时，脚本会自动弹出一个配置窗口，如下图所示（示例）：
![海投机界面](..\static\img\autosend\autosend-step4.png)

### 🧭使用步骤

输入职位关键词（例如：前端实习 / 产品经理）

填写你想求职的城市（例如：上海 / 深圳）

点击[🚀一键出击，去敲门] 

脚本将自动执行投递动作，并在海投机界面输出日志

:::warning 注意

使用脚本时，请保持页面前台打开，避免浏览器休眠。

脚本仅限个人辅助使用，请勿用于商业用途。
:::

## 💬 常见问题（FAQ）

<details>
<summary>1️⃣ 为什么打开 boss职位界面后 海投机窗口未弹出？</summary>

请确认以下几点：

* ✅ 浏览器已安装并启用了 **Tampermonkey（篡改猴）**；
* ✅ 脚本已成功启用（图标应为亮色猴子）；
* ✅ 浏览器扩展已启用开发者模式。

💡 **解决方法**：
重新刷新页面或在篡改猴面板中手动启用脚本。

</details>
