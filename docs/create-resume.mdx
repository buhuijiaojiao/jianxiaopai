# 🧩 如何创建简历

在 **简小派平台** 创建一份专业简历，是求职成功的第一步。  
本指南将带你了解最新版本的简历创建流程。

---

## 🚀 创建步骤

### 1️⃣ 进入简历管理界面

在左侧导航栏中点击 **「简历」**，即可进入 **简历管理界面**。  
此处你可以查看、管理、编辑和导出所有的简历。

![进入简历管理界面](/img/create-resume/create-resume-step1.png)

---

### 2️⃣ 创建新的简历

点击简历卡片右上角的 **「创建新的」** 按钮。  

系统将为你提供多种创建方式：

选择合适的方式后，即可进入简历编辑界面。  
![创建新的简历](/img/create-resume/create-resume-step2.png)
![按需选择创建方式](/img/create-resume/create-resume-step3.png)

---

### 3️⃣ 编辑与完善内容

进入编辑界面后，你可以填写或修改以下内容：

- 个人信息（姓名、联系方式、邮箱）  
- 教育背景与工作经历  
- 技能特长与项目经验  
- 职业目标与自我介绍  


![编辑简历](/img/create-resume/create-resume-step4.png)

---

### 4️⃣ 保存与预览简历

填写完成后，系统会自动保存你的修改。  

* 简历编辑于优化 ：请参阅简历优化文档。

你可以点击右上角 **「导出PDF」** 按钮，以PDF格式导出。

![导出简历](/img/create-resume/create-resume-step5.png)
