# 🩺 智能体检

**智能体检** 是简小派简历优化流程的第一步。  
系统会对简历进行 **深度分析评分**，并生成针对性优化建议，帮助你快速发现问题并提升通过率。

---

## 🚀 使用步骤

### 1️⃣ 进入「简历管理」界面
在左侧导航栏点击 **「简历」**，进入简历管理界面。  
选择需要体检的简历，进入简历详情页。

![简历诊断步骤1](/img/optimize-resume/resume-diagnosis/resume-diagnosis-step1.png)

---

### 2️⃣ 点击「简历诊断」按钮
在简历详情页侧边栏点击 **「简历诊断」** 按钮，然后点击开始诊断。

![简历诊断步骤2](/img/optimize-resume/resume-diagnosis/resume-diagnosis-step2.png)

---

### 3️⃣ 填写求职信息表单（可跳过）

体检开始前，系统会弹出 **求职信息表单**，用于了解你的求职方向，以便生成更精准的分析结果。

表单内容如图所示：

你可以选择：
- ✅ **填写后开始诊断**（推荐） → 系统将根据目标岗位智能分析匹配度  
- ⏭ **跳过填写** → 直接进入通用简历诊断流程  

![求职信息表单](/img/optimize-resume/resume-diagnosis/resume-diagnosis-step3.png)

---

### 4️⃣ 阅读体检报告
等待体检完成后，系统会展示一份简历分析报告，包括：

- **AI诊断报告**（匹配度 / 完整度 / 专业度）  
- **优化建议**：针对教育经历、项目描述、技能关键词等模块给出具体改进方向  

请自行选择对应区域进行修改。

![体检报告](/img/optimize-resume/resume-diagnosis/resume-diagnosis-step4.png)

---

### 5️⃣ 修改并复检
阅读完成并修改后，可以点击右上角的重新诊断进行复检

系统会重新执行体检流程并生成最新评分。  
多次复检可以帮助你验证优化效果。

---

## 🎯 优化指引

- **提示“项目经历描述不够丰富”** → 前往 **[项目工坊]** 自动生成结构化项目内容。  
- **提示“缺乏数据化成果”** → 使用 **[小派优化]** 优化表达，提升专业度与可读性。

## ⚠️ 注意事项

:::tip 温馨提示
- 智能体检得分仅供参考，重点在真实、逻辑清晰的内容。  
- 避免堆砌无意义关键词，过度优化反而会影响阅读体验。  
- 每次修改后建议重新体检，验证优化方向是否正确。
:::
